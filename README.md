lineupengine [![NPM version][npm-image]][npm-url]
=====================


Development Environment
-----------------------

**Installation**

```bash
git clone https://github.com/sgratzl/lineupengine.git
cd lineupengine
npm install
```

**Build distribution packages**

```bash
npm run build
```

**Run Linting**

```bash
npm run lint
```


**Watch file changes**

```bash
npm run watch
```
